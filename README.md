**StudyHub** is a comprehensive, open-source platform designed to help SPPU (Savitribai Phule Pune University) engineering students access high-quality study materials, including Previous Year Questions (PYQs), handwritten notes, decoded books, and video lectures.

üöÄ **Live Demo:** [Click here to view on Vercel](https://study-material-website-cyan.vercel.app/)

## ‚ú® Features

- **Subject-wise Resources**: Organized content for various branches (Computer Science, ENTC, IT, etc.) and semesters.
- **Dynamic Resource Handling**: Smart routing to fetch and display resources for specific subjects.
- **Previous Year Questions (PYQs)**: extensive collection of past exam papers.
- **Handwritten Notes**: Curated notes from toppers and experienced students.
- **Contribution System**: Easy way for students to upload and share their own resources via Google Forms or Telegram.
- **Responsive Design**: Beautiful, modern UI built with Tailwind CSS, fully optimized for mobile and desktop.
- **State Persistence**: Remembers your selected Branch and Semester so you don't have to restart every time.

## üöÄ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/study-material-website.git
    cd study-material-website/study-material
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) in your browser.
